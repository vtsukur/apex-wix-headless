# TH-12 video stills

Source stills for generating the TH-12 lap-video hero with Kling (image-to-video).
Not used by the site build — reference assets only.

All images are AI-generated (Wix AI / Runware, `google:4@2`) **original fictional
car designs** — unbranded, no real-marque badges, logos, or plates. Stills A and B
were generated with `reference-rear-34.png` passed as a reference image so the car
is identical across angles.

## Shared look (critical for intercutting)

Overcast dusk · quiet empty circuit · dry grey tarmac · low kerbs/barriers ·
muted flat light · same paint/wheels. Nose points **right** in every exterior
still, so the car travels left→right through the lap.
